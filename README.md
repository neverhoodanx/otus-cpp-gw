# otus-cpp-gw
Graduation qualifying work on the topic "Messenger - Instant Messaging Service"
---

## Requirements
 - conan 2.0.0
 - python 3.9
 - cmake 3.24 

## Usage
```sh 
$ message_service <port>
```
## Build
```sh
$ make build # Build images
$ make run   # Start container
$ make logs  # View logs
$ make clean # Remove containers and images
```
